# Development Roadmap

## Phase 1: Foundation

- Next.js App Router scaffold
- shadcn/ui design system
- Prisma schema
- public website shell
- lead validation and routing engine
- CRM dashboard shell

## Phase 2: Persistence

- database migrations
- NextAuth credentials/provider setup
- server actions for leads, notes, follow-ups, and CMS resources
- email notifications through Resend
- upload pipeline through S3-compatible storage

## Phase 3: Operations

- role permissions
- audit logs
- export CSV/Excel
- analytics dashboards
- settings UI for lead routing and website content

## Phase 4: Launch

- Docker image
- VPS deployment
- backup and restore process
- rate limiting
- SEO verification
- Lighthouse and accessibility pass
