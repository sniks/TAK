# SEO Strategy

- Generate metadata for all public routes with title, description, canonical URL, Open Graph, and Twitter cards.
- Generate service-specific landing pages for all service categories.
- Use JSON-LD for Organization, LocalBusiness, Service, Article, BreadcrumbList, and WebSite search action.
- Keep public pages server-rendered where possible for crawlability.
- Generate `sitemap.ts` and `robots.ts`.
- Optimize images with `next/image`, explicit dimensions, and descriptive alt text.
- Track Core Web Vitals and keep Lighthouse above 90 by avoiding oversized scripts and lazy-loading non-critical media.
